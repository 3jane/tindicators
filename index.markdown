## Quick start
```bash
pip install tindicators
```
```python
from tindicators import ti
```

Example usage:
```python
>>> import numpy as np
>>> ti.sma(np.arange(10), 3)
array([nan, nan,  1.,  2.,  3.,  4.,  5.,  6.,  7.,  8.])
```
